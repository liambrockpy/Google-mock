# Google-mock

## Installation & Usage

### Installation

- Clone or download the repo.
- Navigate to `server` folder.
- Run `npm install` to install dependencies.

### Usage

- Run `npm start` to launch server.
- Run `npm test` to launch test suite.
- Run `npm run coverage` to create coverage report.
